# Rename multiple files
___Постановка задачи:___    
Есть 200 файлов вида "nm-XXX.pdf", где ХХХ - порядковый номер, который трогать не надо.    
Эти файлы надо переименовать в "name #XXX.pdf".

- [ ] Возможность работы из консоли
